sequenceDiagram
    participant User as User/API
    participant FP as FlexPrice Backend
    participant Stripe as Stripe API
    participant Customer as Customer Browser
    participant Webhook as Webhook Handler
    
    Note over User,Webhook: STEP 1: Payment Link Creation
    
    User->>FP: CreatePaymentLink(invoice_id, amount, customer_id)
    
    FP->>FP: 1. Get Invoice Details
    FP->>FP: 2. Validate Invoice Status (not paid/voided)
    FP->>FP: 3. Validate Amount ≤ Remaining Balance
    FP->>FP: 4. Validate Currency Matches
    
    FP->>FP: 5. EnsureCustomerSyncedToStripe()
    Note right of FP: Check metadata["stripe_customer_id"]<br/>If not found, create in Stripe
    
    FP->>Stripe: GET Customer or CREATE Customer
    Stripe-->>FP: stripe_customer_id
    
    FP->>FP: 6. Build Line Items from Invoice
    Note right of FP: Product name, description,<br/>amount, currency
    
    FP->>FP: 7. Create Metadata
    Note right of FP: flexprice_invoice_id<br/>flexprice_payment_id<br/>customer_id<br/>payment_source: "flexprice"<br/>payment_type: "checkout"
    
    FP->>Stripe: CREATE Checkout Session
    Note right of FP: Mode: "payment"<br/>Customer: stripe_customer_id<br/>LineItems: invoice details<br/>SuccessURL & CancelURL<br/>SaveCardAndMakeDefault flag
    
    Stripe-->>FP: Checkout Session {id, url, payment_intent_id}
    
    FP->>FP: 8. Create Payment Record (status: pending)
    FP-->>User: Return Payment URL
    
    Note over User,Webhook: STEP 2: Customer Payment
    
    User->>Customer: Redirect to Payment URL
    Customer->>Stripe: Opens Checkout Page
    Customer->>Stripe: Enters Card Details
    Customer->>Stripe: Submits Payment
    
    Stripe->>Stripe: Process Payment
    Stripe->>Stripe: Create Payment Intent
    Stripe->>Stripe: Charge Card
    
    Note over User,Webhook: STEP 3: Webhook Processing
    
    Stripe->>Webhook: checkout.session.completed
    
    Webhook->>Webhook: Parse Event Data
    Webhook->>Webhook: Extract flexprice_payment_id from metadata
    
    Webhook->>FP: GetPayment(flexprice_payment_id)
    FP-->>Webhook: Payment Record
    
    Webhook->>Webhook: Check if already succeeded
    
    alt Payment Not Yet Succeeded
        Webhook->>Stripe: Get Full Payment Intent Details
        Stripe-->>Webhook: PaymentIntent with payment_method_id
        
        Webhook->>FP: HandleFlexPriceCheckoutPayment()
        
        FP->>FP: Update Payment Status = "succeeded"
        FP->>FP: Set gateway_payment_id = payment_intent_id
        FP->>FP: Set payment_method_id
        
        alt SaveCardAndMakeDefault = true
            FP->>Stripe: Set as Default Payment Method
            Note right of FP: Update customer.invoice_settings<br/>.default_payment_method
        end
        
        FP->>FP: ReconcilePaymentWithInvoice()
        Note right of FP: Calculate new amounts:<br/>amount_paid += payment_amount<br/>amount_remaining = due - paid<br/>Determine payment_status
        
        FP->>FP: Update Invoice
        Note right of FP: payment_status = succeeded/pending/overpaid<br/>amount_paid<br/>amount_remaining
        
        opt Stripe Invoice Exists
            FP->>Stripe: AttachPaymentToStripeInvoice()
            Note right of FP: Links PaymentIntent to Stripe Invoice
        end
        
        Webhook-->>Stripe: 200 OK
    else Payment Already Succeeded
        Webhook-->>Stripe: 200 OK (skip processing)
    end
    
    Note over User,Webhook: ✅ COMPLETE
    
    FP->>User: Webhook Event (optional notification)
